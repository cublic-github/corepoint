# CorePoint プロジェクト現状整理

## 1. プロジェクト概要

**CorePoint** は、ユーザーの認知特性を5次元ベクトルで可視化し、ターゲット（理想像）との適合率（シンクロ率）を算出するアセスメントシステム。

### コアモデル: V-Sync (Vectorized Sync)

ユーザーの特性を `vec = [c1, c2, c3, c4, c5]`（各 0.0〜10.0）で表現する。

| 成分 | 名称 | 低値 | 高値 |
|------|------|------|------|
| c1 | 抽象化レイヤー | 具体的・事実ベース | 構造的理解・メタ認知 |
| c2 | 指向性・遊び心 | 収束的・正論重視 | 発散的・ユーモア |
| c3 | 合理的リアリズム | 情緒的・こだわり | 損得・効率重視 |
| c4 | 社会的距離 | 傍観・ドライ | 介入・お節介 |
| c5 | 言語変換解像度 | 専門用語そのまま | 相手に合わせた翻訳 |

### アルゴリズム

- **距離計算:** 重み付きユークリッド距離 `D = sqrt(Σ w_i * (c_Ui - c_Ti)^2)`
- **シンクロ率:** `Score = 100 / (1 + 0.1 * D)`
- **減衰式:** `NewValue = CurrentValue + Δ * (1 - |CurrentValue - 5| / 5)`（端に近づくほど変化を抑制）

### 判定ラベル

| スコア | ラベル | 状態 |
|--------|--------|------|
| 90-100% | Synchronized | 阿吽の呼吸 |
| 70-89% | Resonant | 共鳴 |
| 50-69% | Strained | 要歩み寄り |
| 30-49% | Disconnected | 断絶 |
| 0-29% | Void | 虚無 |

## 3. 作成済みHTMLモックの内容

### 3-1. 回答画面 (`mock/quiz.html`)

回答者（候補者）がアクセスするクイズ画面。

- **ヘッダー:** CorePointロゴ
- **プログレスバー:** `Question N / 全問数` + パーセント表示
- **設問カード:** カテゴリラベル + 質問テキスト
- **選択肢:** A〜E の5択カード（クリックで選択状態に切り替え）
- **「次の質問へ」ボタン:** 選択後に有効化
- **サンクス画面:** 全問回答後に表示される完了メッセージ

**サンプル設問:** 「タメ口の語源は何だと思う？」（concept.md の c1〜c5 スコアリング例に対応）

### 3-2. 管理画面 (`mock/dashboard.html`)

管理者が使うダッシュボード。3つのタブで構成。

#### タブ1: 回答者一覧
- 検索バー（名前・メール）
- グローバル基準プリセット（ターゲット人物像 + 重みプリセット）のドロップダウン
- ユーザーテーブル（名前、メール、回答日、Sync Rate、判定ラベル）
- ソート機能（回答日 / Sync Rate）
- **ユーザー詳細パネル**（右スライド）:
  - シンクロ率リングチャート（SVG）
  - プリセット切り替え（個別ユーザー単位）
  - 5次元ベクトルのバーチャート（ターゲット位置のマーカー付き）
  - Target Comparison 表示

#### タブ2: クイズ管理
- **使用中のクイズ:** ドラッグで並び替え、編集・ストック戻しボタン
- **登録済みストック:** 未使用の設問一覧、「+使用する」で有効化
- **JSON一括インポート:** テンプレート付き（question + choices[5] with vector）
- **新規作成ボタン**

#### タブ3: 基本設定
- **ターゲット人物像プリセット管理:**
  - 分析者タイプ: `[8.0, 2.0, 8.0, 3.0, 6.0]`
  - エンタメタイプ: `[4.0, 9.0, 3.0, 7.0, 8.0]`
  - メンタータイプ: `[7.0, 4.0, 5.0, 9.0, 9.0]`
  - リアリストタイプ: `[5.0, 1.0, 9.0, 2.0, 4.0]`
  - バランスタイプ: `[5.0, 5.0, 5.0, 5.0, 5.0]`
- **重みプリセット管理:**
  - 均等ウェイト: `[1, 1, 1, 1, 1]`
  - 知的対話重視: `[5, 1, 1, 1, 5]`
  - ノリ・相性重視: `[1, 5, 1, 5, 2]`
- 各プリセットの追加・編集・削除
